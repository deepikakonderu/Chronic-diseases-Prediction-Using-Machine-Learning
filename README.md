# Chronic Diseases Prediction Using Machine Learning

## 📌 Project Overview

This project is a **web-based application** that predicts the likelihood of multiple chronic diseases using **Machine Learning techniques**.

The system analyzes patient medical data and predicts diseases such as **Diabetes, Stroke, Chronic Kidney Disease, Hypertension, and Cardiovascular Disease**.

The application is built using **Python and the Django framework** and provides a simple web interface where users can enter health parameters and receive disease predictions.

---

## 🚀 Features

* Predicts multiple chronic diseases
* User-friendly web interface
* Machine Learning-based prediction system
* Uses real-world medical datasets
* Fast and accurate predictions

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/deepikakonderu/Chronic-diseases-Prediction-Using-Machine-Learning.git
```

### 2️⃣ Navigate to the project directory

```
cd Chronic-diseases-Prediction-Using-Machine-Learning
```

### 3️⃣ Install required dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the Django server

```
python manage.py runserver
```

### 5️⃣ Open the application in your browser

```
http://127.0.0.1:8000/
```
